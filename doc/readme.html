功能说明

  • WebCrawlerII:
    • 一般的网页源码抓取
      • 用HttpURLConnection抓取
      • 用HttpClient抓取
    • 网页链接匹配
      • 用正则表达式匹配href=
      • 用Jsoup匹配a[href]
    • 网页图片下载
      • 用正则表达式匹配.jpg/bmp/gif
      • 用Jsoup匹配img[src]
    • 规定大小范围的网页图片下载

  • WebContentCrawlerII:

    • 继承自WebCrawlerII，故可实现其父类的所有功能
    • 网页正文源码抓取
    • 网页正文文本抓取
      • 用正则表达式匹配

    • 网页图片输出
      • Html格式
      • 一般格式
    • 网页正文输出
      • Plain Text格式
      • Html格式

  • 具体说明在readme.pdf中
